# Ron Golan — Portfolio

My personal portfolio site. I'm a mobile developer (Android & iOS) and a
fourth-year Computer Science student at Afeka College of Engineering,
graduating August 2026.

**Live site:** https://rongo270.github.io/my-portfolio

## About

I design, build and publish complete mobile apps — taking each one from
idea to release on the App Store and Google Play. My main stack is Kotlin
and Android Studio on Android, and Swift, SwiftUI and Xcode on iOS, backed
by Firebase and Firestore.

The site walks through who I am, my featured app **CheckChange** (live on
both stores), what I'm working on, my tech stack, and how to get in touch.

## Built with

- React (Create React App)
- Bootstrap
- AOS (scroll animations)
- react-typed
- Deployed to GitHub Pages via `gh-pages`

## Running locally

```bash
npm install
npm start
```

The app runs at http://localhost:3000.

## Other scripts

```bash
npm run build    # production build
npm run deploy   # build and publish to GitHub Pages
```
